# Usage

## Required

- Ruby 2.7.2 or later
- Bundler
- Homebrew

## Install

```
brew bundle
bundle install
```

## Execution

### `001_zlib`

This script outputs binary. Redirect to PNG file to view as image.

```
ruby 001_zlib/main.rb > tmp/001_zlib.png
```

### Cairo scripts

```
ruby 002_cairo/hinomaru.rb
ruby 003_cairo_fractal/carpet.rb
ruby 003_cairo_fractal/snowflake.rb
```
